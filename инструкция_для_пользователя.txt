ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ БОТА ДЛЯ РАСЧЕТА МЕБЕЛИ

КАК РАБОТАТЬ С БОТОМ:

1. Найди бота в Telegram и нажми /start

2. Отправь боту .docx файл с таблицей изделий

   Таблица должна содержать столбцы:
   - Название изделия
   - Визуал (фото проекта - опционально)
   - Размеры (ширина x высота x глубина в мм)
   - Корпус (материал корпуса)
   - Фасад (описание фасада)

3. Бот автоматически:
   - Извлечет данные из таблицы
   - Определит материал фасада (ЛДСП/Эмаль/Шпон)
   - Определит наличие двухсторонней отделки
   - Определит наличие фрезеровки
   - Рассчитает стоимость корпуса и фасадов
   - Создаст коммерческое предложение в формате PPTX

4. Получишь:
   - Текстовое сообщение с расчетом
   - PPTX презентацию с фото и описанием

ПРИМЕРЫ ОПИСАНИЯ ФАСАДОВ:

Для правильного распознавания используй ключевые слова:

Материалы:
- "МДФ эмаль" или "эмаль" или "краска" → Эмаль
- "МДФ шпон" или "шпон" → Шпон
- "ЛДСП" или без указания → ЛДСП

Двухсторонняя отделка:
- "2х стор", "2-стор", "двухсторонняя"

Фрезеровка:
- "фрез", "фрезеровка", "классика", "рисунок"

Примеры:
- "МДФ эмаль 2х стор фрез" → Эмаль, двухсторонняя, с фрезеровкой
- "МДФ шпон" → Шпон, односторонний, без фрезеровки
- "ЛДСП" → ЛДСП, односторонний, без фрезеровки

РАСЧЕТ СТОИМОСТИ:

Корпус (ЛДСП):
- Базовая цена: 24,000 руб/м2
- Корректируется по глубине: (Ш × В) × Цена × (Глубина / 600)

Фасады:
- ЛДСП: 6,500 руб/м2 (удваивается при 2х стор)
- Эмаль: 17,000 руб/м2 (+12,000 за 2х стор, +8,000 за фрезу)
- Шпон: 30,000 руб/м2 (+12,000 за 2х стор, +8,000 за фрезу)

ИЗМЕНЕНИЕ ЦЕН:

Чтобы изменить цены, отредактируй файл data/prices.json:

{
  "body_materials": {
    "egger": 24000,
    "lamarty": 11000
  },
  "facade_materials": {
    "egger": 6500,
    "enamel": 17000,
    "veneer": 30000
  },
  "surcharges": {
    "two_sided": 12000,
    "milling": 8000
  }
}

После изменения перезапусти бота.

ПОДДЕРЖКА:

Если бот не распознает материал или параметры:
1. Проверь правильность написания ключевых слов
2. Убедись, что в таблице правильные столбцы
3. Проверь формат размеров (должны быть числа)
