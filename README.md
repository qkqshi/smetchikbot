# Smetchikbot - Генератор коммерческих предложений для мебели

Telegram-бот для автоматической генерации коммерческих предложений (КП) на основе данных из Word документов.

## Возможности

- 📄 Парсинг Word документов (.docx) с таблицами изделий и мебели
- 📊 Генерация КП в трех форматах: PDF, PowerPoint (PPTX), Excel (XLSX)
- 🖼️ Автоматическое извлечение и добавление изображений из документов
- 💰 Автоматический расчет стоимости на основе прайс-листа
- 📱 Удобный интерфейс через Telegram бота
- 🎨 Профессиональные шаблоны с обложками и финальными слайдами

## Структура проекта

```
smetchikbot/
├── src/
│   ├── bot.py              # Telegram бот
│   ├── parser.py           # Парсер Word документов

│   ├── pdf_generator.py    # Генератор PDF
│   ├── pptx_generator.py   # Генератор PowerPoint
│   ├── excel_generator.py  # Генератор Excel
│   └── main.py            # Точка входа
├── data/
│   └── prices.json        # Прайс-лист материалов
├── templates/             # Шаблоны обложек и финальных слайдов
├── outputs/              # Сгенерированные файлы
└── uploads/              # Загруженные документы

```

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/qkqshi/smetchikbot.git
cd smetchikbot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` с токеном бота:
```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

4. Добавьте шаблоны обложек в папку `templates/`:
   - `cover.png` - горизонтальная обложка для PPTX
   - `cover_vert.png` - вертикальная обложка для PDF
   - `end1.png`, `end2.png`, `end3.png` - финальные слайды

## Использование

### Запуск бота

```bash
python src/main.py
```

### Формат входного документа

Word документ должен содержать:

1. **Таблицу изделий** с колонками:
   - Наименование
   - Размеры (ШхГхВ)
   - Материал корпуса
   - Описание фасадов
   - Количество
   - Дополнительная информация

2. **Таблицу мебели** (опционально) с колонками:
   - Наименование
   - Количество
   - Цена за единицу
   - Общая стоимость

3. **Изображения** - автоматически извлекаются из документа

### Команды бота

- `/start` - Начать работу с ботом
- Отправьте Word документ (.docx) для генерации КП
- Опционально: отправьте номер телефона для добавления на обложку

## Формат выходных файлов

### PDF
- Вертикальная обложка с номером телефона
- Отдельная страница для каждого изделия/мебели с фото и описанием
- Итоговая таблица со всеми позициями
- Примечание с условиями (+10% услуги, -3% за наличные, срок 45 дней)
- Финальные страницы

### PowerPoint (PPTX)
- Горизонтальная обложка с номером телефона
- Отдельный слайд для каждого изделия/мебели с фото
- Итоговая таблица на отдельном слайде
- Примечание с условиями
- Финальные слайды

### Excel (XLSX)
- Детальная таблица всех позиций
- Автоматический расчет итоговой стоимости
- Форматирование для печати

## Прайс-лист

Файл `data/prices.json` содержит цены на материалы:

```json
{
  "ЛДСП": {
    "price_per_sqm": 1500,
    "edge_price_per_m": 50
  },
  "МДФ": {
    "price_per_sqm": 2500,
    "edge_price_per_m": 80
  }
}
```

## Примечания в КП

Автоматически добавляется под итоговой таблицей:
- Услуги упаковка, доставка, подъем, сборка и монтаж, сбор мусора +10% к стоимости
- -3% Скидка за наличные
- Срок 45 рабочих дней
